# Character Images

Place your character illustrations for different clothing categories in this directory.

## Required Images

1. `winter_cold.png` - Character outfit for very cold weather (≤0°C)
2. `winter_mild.png` - Character outfit for cool weather (1-10°C)
3. `summer_hot.png` - Character outfit for hot weather (≥30°C)
4. `summer_warm.png` - Character outfit for warm weather (25-29°C)
5. `rain.png` - Character outfit for rainy weather (high humidity)
6. `windy.png` - Character outfit for windy conditions
7. `normal.png` - Character outfit for pleasant weather
8. `gym.png` - Character outfit for indoor running (when outdoor conditions are unsafe)

## Image Specifications

- **Recommended size**: 600x800px (or maintain 3:4 aspect ratio)
- **Format**: PNG with transparency preferred
- **Style**: Clean, cartoon character illustrations
- **Content**: Character wearing appropriate running clothing for each condition

## Tips

- Keep the character consistent across all images
- Make sure the clothing is clearly visible
- Use bright, cheerful colors to match the app's design
- Consider the background (transparent PNGs work best)
